# Smart Expense System

A full-stack expense management application built using Spring Boot, React, MySQL, and JWT Authentication.

## Features

- User Registration & Login
- JWT Authentication
- Expense Management
- Budget Management
- Dashboard Analytics
- Secure REST APIs

## Technologies Used

### Backend
- Java
- Spring Boot
- Spring Security
- JWT
- JPA/Hibernate
- MySQL

### Frontend
- React
- Axios
- Bootstrap
- React Router

## Project Structure

Frontend:
React + Vite

Backend:
Spring Boot REST API

Database:
MySQL
