# Smart Expense System

A Full Stack Expense Management Application developed using Spring Boot, React, MySQL, JWT Authentication and REST APIs.

## Features

- User Registration and Login
- JWT Authentication
- Expense Management
- Budget Management
- Dashboard Analytics
- Expense Tracking
- RESTful APIs
- Secure Authentication

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- JWT
- Hibernate
- JPA
- MySQL

### Tools
- Maven
- Git
- GitHub
